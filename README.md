# Power-BI-Sales-Dashboard
This Power BI dashboard provides a comprehensive analysis of sales data, helping business stakeholders track performance across different regions, cities, and product categories. It transforms raw data into actionable insights for better decision-making.
📊 Sales Performance Dashboard (Power BI)
📝 Project Overview:This Power BI dashboard provides a comprehensive analysis of sales data, helping business stakeholders track performance across different regions, cities, and product categories. It transforms raw data into actionable insights for better decision-making.
Dashboard Preview:https://github.com/jweel-hossain/Power-BI-Sales-Dashboard/commit/7ad2122c9e318bdf1e2bc14e5e40b09713fffc7c
💡 Key Insights & FeaturesSales Trends:
Analyzed total sales ($2M) and identified the top-performing products and regions.Geographical Analysis: Used a Map Visual to track sales distribution across different States.Top Performers: Created a Top 10 Cities bar chart to focus on high-revenue locations.Interactive Filters: Integrated slicers for Ship Mode, Customer Segment, and Category for a customized view.
🛠️ Tools & Technologies Used-Power BI Desktop: For data visualization and dashboard building.
Power Query: For data cleaning and transformation.
DAX (Data Analysis Expressions): Used for calculating measures like Total Sales and Profit.
Dataset: Sample Superstore/Sales Data.
🚀 How to ViewDownload the .pbix file from this repository.Open it using Power BI Desktop.
